# CV Web Personal - Angie Katherin Arboleda Gonzalez

## Descripción
Proyecto personal: página web que muestra información académica y profesional de Angie Katherin Arboleda Gonzalez.  
Incluye secciones de “Sobre mí”, formación académica, habilidades técnicas, proyectos, objetivo profesional y contacto.  

## Tecnologías utilizadas
- HTML
- CSS

## Cómo abrir el proyecto
1. Abrir `index.html` directamente en cualquier navegador web (Chrome, Edge, Firefox, etc.).  
2. Recomendado: usar **Live Server** en VS Code para abrirlo con una URL local y ver cambios en tiempo real:
   - Se abrirá algo como `http://127.0.0.1:5500/index.html`
   - Funciona en cualquier navegador.


## Contenido de la página
- **Header**: Nombre y título profesional con estilo moderno.  
- **Sobre mí**: Foto pequeña y descripción personal.  
- **Formación Académica**: Estudios y materias cursadas.  
- **Habilidades Técnicas**: Conocimientos en HTML, CSS, JavaScript y SQL.  
- **Proyectos**: Ejemplo de CV web personal.  
- **Objetivo Profesional**: Objetivos y aspiraciones en tecnología.  
- **Contacto**: Email y perfil de LinkedIn.

## Notas adicionales
- Todos los archivos del proyecto (HTML, CSS e imagen) se encuentran en esta carpeta o sus subcarpetas (`imagenes/`).  
- Este proyecto es ideal para mostrar como **primer proyecto web personal** en un portafolio profesional.x
